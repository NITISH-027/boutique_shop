# Rufes Fashions image folders

Drop your local boutique photos into these folders and redeploy/restart the Next.js app. No code changes are required for gallery and lookbook additions.

- `hero-boutique.jpg` — hero feature image
- `about-store.jpg` — about section image
- `collections/` — collection images. Recommended names: `bridal-wear.jpg`, `party-wear.jpg`, `custom-stitching.jpg`, `ethnic-collections.jpg`, `kids-designer-dresses.jpg`
- `gallery/` — unlimited gallery images; supported: jpg, jpeg, png, webp, avif
- `lookbook/` — unlimited horizontal lookbook images; supported: jpg, jpeg, png, webp, avif
